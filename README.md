# Portfolio 2026 — Neliuzx

Interactive portfolio built with React, Three.js and a little bit of GSAP.

## Concept

A 3D cube acts as the main navigation. Each face opens a different section — Projects, Skills, GitHub, About, Contact.

## Stack

- React + Vite
- Three.js
- GSAP
- Vanilla CSS

## Run locally

```bash
npm install
npm run dev
```

## Debug panel

Access the debug panel via `Cmd/Ctrl + Shift + K` or by adding `#debug` to the URL.
